# Content Authoring

Want to create a DDDnD content pack? This guide explains how to structure scenarios, cards, stakeholders, and rules.

## Content Pack Structure

A content pack is a directory with:

- `manifest.json` - Pack metadata and inventory
- `scenarios/` - Scenario definitions
- `cards/` - Card definitions
- `stakeholders/` - Stakeholder definitions
- `stakeholder-reaction-rules/` - Rules for how stakeholders react
- And other content types (events, delayed effects, classes, etc.)
