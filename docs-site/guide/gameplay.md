# Gameplay

## Core Loop

The game is played over a series of turns. Each turn you select a card to play, resolve its effect, and then update stakeholder satisfaction based on the outcome.

### Win Conditions

- **Success**: Achieve your target outcome and maintain stakeholder satisfaction
- **Partial Success**: Meet some objectives but not all
- **Failure**: Fail to achieve core objectives
