# UI Surface Assets

This folder is reserved for reusable presentation-layer surface artwork such as panel textures, separators, and decorative chrome.

Structure:

- frames/ for panel and card frames
- icons/ for UI iconography used as chrome
- badges/ for reusable status/label badges
- ornaments/ for decorative motifs and separators

Rules:

- UI-only usage
- No simulation/domain references
- Register runtime usage through src/ui/config/presentation_asset_registry.ts
